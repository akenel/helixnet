How to Use This Platform

HelixNet runs 4 apps on one platform. Each app has its own login, its own Keycloak realm (or shared realm), and its own set of roles. The top navigation bar on every page lets you jump between apps.

1

Pick an App

Choose the app you need to test from the cards below.

2

Login

Click the login link. Use the test account for the role you want to verify.

3

Test

Follow the testing checklist for that app. Report bugs via Telegram.

4

Switch Apps

Use the platform nav bar at the top to jump between apps.

Password for ALL accounts: helix_pass -- every user, every app, no exceptions.
Language: Camper and ISOTTO default to Italian. Add ?lang=en to any URL for English.
Certificate warning: First visit will show a security warning (self-signed cert). Accept it once -- it's safe.
