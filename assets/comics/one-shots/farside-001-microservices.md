## DRAW NOTES

1. **The Wall**: Should look OVERWHELMING
   - Boxes connected by lines everywhere
   - Labels like "auth-service-v2", "user-gateway-prod", "cache-invalidator-helper"
   - Some boxes have warning symbols
   - Lines crossing everywhere (spaghetti)

2. **The CTO**: Proud, chest out
   - Pointer or hand gesturing at wall
   - Sponsor patches visible (SAP, KPMG, Accenture)
   - Doesn't see the kid

3. **The Kid**: Simple, small, correct
   - Three LEGO blocks stacked neatly (red, blue, yellow)
   - Calm expression
   - Already solved it

4. **The Contrast**: Visual joke
   - Complexity vs simplicity
   - Size vs effectiveness
   - Pride vs actual solution
