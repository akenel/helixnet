When a product can't be found locally (catalog / reference), a 🔎 Search the web button opens Google pre-filled with the barcode + name in a new tab — your manual "go to Google" move, one tap. This tests the button appears on every miss and searches the right thing. Guest never sees it — staff only.

✅ Proven live: barcode 7610442291046 → "Sun Snack" via a barcode DB. Tier 1 (this button) opens Google; Tier 2 (in-app auto-lookup) is banked for a service pick.
