1 · Cash — type the real amount & stack notes the one you asked for

Nobody pays exact. The Swiss hand over a 50 for a pack of gum, or 75.00 for a 72.50 bill to get clean coins back — and combine notes. You should never have to fake "exact" again.

  1. Ring a small cart, choose Cash. an "Amount received (CHF)" box appears + note buttons +5 +10 +20 +50 +100 +200 + EXACT + CLEAR
  2. Type an over-amount into the box (e.g. total 9.35 → type 20). Change updates live to the right number (10.65)
  3. Type an odd amount the Swiss way — bill 9.35, type 10.35 (they want a clean 1.00 back). Change = 1.00
  4. Stack notes: tap +50 then +20. Amount received builds to 70.00 (a fifty and a twenty)
  5. Hit CLEAR, then EXACT. CLEAR zeroes it; EXACT sets it to the total, change 0.00
  6. Complete a cash sale with a typed over-amount. rings through; change shown; receipt correct (safe — it's sandbox)
