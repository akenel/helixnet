## For the card-reader ID (what makes a photo useful)
To identify the model + whether it's Bluetooth, the most useful shots are:
1. The **label on the back/bottom** of each reader — model number + brand (e.g. SumUp, Worldline/SIX, myPOS, Nexi, Ingenico, Verifone).
2. The **front face** (screen + logo) of each.
3. Any **box, charging cradle, or sticker** with a model/serial.

Tigs will read these and report: brand, model, likely connectivity (BT/USB/WiFi/SIM), and whether an
integration path exists (many SME readers are standalone-only; some have a cloud/API or BT SDK).
