BL-16 Type the quantity felix

Cashiers were scanning water 6× for six bottles. Now the count on the cart line is typeable.

▶ Open /pos/scan/pos/scan
  1. Search any product → Add to the cart.
    it appears as a cart line with a −/number/+ row.
  2. Tap the number box between − and + and type 6.
    line total = price × 6; it doesn't reset to 1.
  3. Type 0 or gibberish, then tap away.
    snaps back to 1 (removal is the 🗑, not a typo).

BL-21/22 The Order Book ralph seeded

Ralph's reorder pencil-list: what to order, pick which supplier per line, track to-order → on-order → received. It suggests, you decide.

▶ Open Order BookDashboard → 📓 Order Book · /pos/reorder
  1. Open it — the board isn't empty.
    3 lines in To order (incl. 🙋 "Marco wants…") + 1 in On order (Juicy Jays).
  2. In Add: type a product name, pick qty, pick a supplier (dropdown = 7 clean suppliers), Restock, ➕ Add.
    the new line shows in To order with your supplier.
  3. BL-22: on a To-order line, change the supplier dropdown (e.g. FourTwenty → Wellauer).
    it sticks — that line now orders from the new supplier.
  4. Hit 🚚 Ordered on a line, then ✅ Received on it.
    it walks To order → On order → Received.
  5. Add a 🙋 Customer order with a note ("Larry wanted the extra thing").
    lands with the customer note shown on the line.
  6. Log in as a cashier (pam): can you see + add, but NOT advance/delete?
    add works; the 🚚/✅/✕ manager controls are hidden. (optional — needs pam's pw)
