# 📺 YouTube Upload Kit — Dream Weavers Season 1

Everything you need, one folder per video. Each folder has the **video file**, an **UPLOAD.md**
(title + description + tags to copy-paste), and any extras (captions, thumbnail).

## Upload in this order
1. **`1-full-movie/`** — the flagship (7:19). Upload this first; everything else points to it.
2. **`2-short-scammer/`** — the Short with the strongest hook ("app catches a scammer"). Bait → the movie.
3. **`3-short-lift/`** — the payoff Short ("20 neighbours, one crane"). Bait → the movie.

## How (each video)
1. studio.youtube.com → **Create → Upload** → pick the `.mp4` in the folder.
2. Open that folder's **UPLOAD.md** → copy the **Title**, **Description**, **Tags** into YouTube.
3. Full movie only: add the **captions.srt** (Subtitles → Add → Upload → With timing → English)
   and the **thumbnail.png**.
4. Audience: **Not made for kids**. Visibility: **Unlisted first** to preview → then **Public**.
5. Add the full movie to the playlist **"Dream Weavers — Mike's Big Move"**.
6. After the movie is live, paste its URL into the two Shorts' descriptions (where it says `[FULL MOVIE LINK]`).

## Tip
Upload the **full movie first**, grab its URL, then do the Shorts so you can drop the real link in.
The scammer Short is your best discovery hook — lead with it.
