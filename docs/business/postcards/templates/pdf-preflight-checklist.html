PDF Pre-Flight Checklist

Run through BEFORE claiming "done" -- NEVER skip this.

1. Source HTML Check

Page Structure
@page { size: A4 portrait; margin: 0; } is set
Page dimensions = 210mm x 297mm (portrait) or 297mm x 210mm (landscape)
All panel heights add up to exactly 297mm
overflow: hidden on .page (no content spill)
For Duplex 3-Card (Format B)
Card size = 137.6mm x 93mm
Gap = 6mm between cards
Grid wrapper at position: absolute; top: 3mm; left: 3mm
Math: 3 + 93 + 6 + 93 + 6 + 93 + 3 = 297mm
NO registration marks on front page (page 1)
Registration marks on back page only (page 2)
Page labels at left: 175mm (invisible in card zone)
For Tent Card
Top tab (42mm) + Front (100mm) + Back (100mm) + Bottom tab (55mm) = 297mm
Front panel has transform: rotate(180deg)
Fold tick marks (8mm) at edges only, not full borders
Content
All placeholder text [BRACKETS] replaced with real content
Business name spelled correctly (verified with client)
Phone numbers correct (call to verify)
QR code image path resolves (not broken)
All image paths resolve (check browser console for 404s)
Italian text proofread (no Google Translate errors)

2. Browser Preview

Opened HTML in Firefox / Chrome
No console errors (F12 > Console)
All images load (no broken image icons)
Text readable at actual size (not too small)
No content overflow or clipping

3. Generate PDF

Command
node /home/angel/repos/helixnet/scripts/postcard-to-pdf.js input.html output.pdf
PDF generated without errors
PDF file size reasonable (photo cards: 500KB-3MB, SVG cards: 50-500KB)
NEVER use wkhtmltopdf or weasyprint. They are uninstalled. Puppeteer (Chrome headless) ONLY.

4. Verify PDF Output

Page Count
Tent card = exactly 1 page
Duplex 3-card = exactly 2 pages
No blank extra pages
Visual Inspection (open PDF in viewer)
Content fills the page -- no large empty gaps
No elements colliding or overlapping
All 3 cards identical on front (for duplex)
All 3 backs identical (for duplex)
White/light backgrounds only (dark kills printers)
QR code sharp and scannable
Text crisp at print size
Print Simulation
Ctrl+P preview matches expected output
Paper size = A4 in print dialog
Scale = 100% (no "fit to page" shrinking)
Tigs Rule: NEVER say "fixed" or "done" without completing steps 1-4. If it overflows to extra pages = NOT done. If there's a blank page = NOT done. If content has gaps = NOT done. Open it. Look at it. Verify it.

5. Delivery

PDF copied to UFA_r2p/ folder
Filename follows convention: duplex-NNN-name.pdf or postcard-client-TENT-FINAL.pdf
Proof sent to client (WhatsApp photo / email)
Client approved design
Printed at ISOTTO (or USB ready for print shop)
Physical card inspected after printing
Source HTML committed to git

Result

ALL CHECKS PASSED -- Safe to print and deliver
