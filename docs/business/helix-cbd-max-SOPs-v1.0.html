SOP-001: Product Sourcing & Supply Chain

Primary Supply Chain Setup SOP-001-A
  1. Contact Felix at 420 Wholesale (Switzerland) for pricing sheet
  2. Review minimum order quantities and shipping terms
  3. Establish payment terms (Net 30 after first order)
  4. Set up recurring order schedule (bi-weekly recommended)

SOP-002: Field Testing Protocol

Standard Field Test Process SOP-002-A
  1. Document test number, date, time, location
  2. Record product details: Strain, Source, Batch, Mix components
  3. Photograph setup (products, rolling materials, environment)
  4. Note pre-test baseline (mood, energy, any symptoms)
  5. Conduct test in appropriate setting (Med shores preferred)
  6. Document at intervals: Ignition, +15min, +30min, +60min
  7. Record observations: Burn quality, ash color, taste, smoothness, effects
  8. Rate overall: 1-10 scale with notes
  9. Upload photos and notes to TigsCave/recon folder
Quality Indicators: White ash = clean burn. Self-extinguishing = no accelerants. Smooth inhale = proper cure.

SOP-003: Postcard Production Pipeline

Photo to Product Pipeline SOP-003-A
  1. Capture raw photos at location (golden hour preferred)
  2. Transfer to /Pictures/palermo-recon/HelixMax - PostCards/
  3. Select best candidates (composition, lighting, landmark value)
  4. Draft NB prompt using vintage travel poster template
  5. Run through Nano Banana AI with prompt
  6. Review output, iterate if needed
  7. Save final as "[Number]. [TITLE] — [Subtitle].jpg"
  8. Design postcard back (standard format with QR code)
  9. Export as print-ready PDF (300dpi, CMYK)
  10. Send to printer with paper specification
Target Turnaround: Photo to print-ready PDF in under 2 hours. Physical product within 72 hours.

SOP-004: Vending Machine Operations

Daily Operations Checklist SOP-004-A
  • Check machine status via remote monitoring
  • Review overnight sales data
  • Verify payment system operational
  • Check inventory levels (restock threshold: 30%)
  • Review any error alerts or customer complaints
  • Plan restocking run if needed
Restocking Protocol SOP-004-B
  1. Generate pick list from inventory system
  2. Pull products from warehouse stock
  3. Verify expiration dates (FIFO rotation)
  4. Transport to machine location
  5. Open machine, document current state
  6. Rotate existing stock forward
  7. Load new stock in rear positions
  8. Verify slot assignments match system
  9. Test dispense on 2-3 random slots
  10. Close and lock machine
  11. Update inventory system

SOP-007: Quality Control & Curation

Curation Philosophy SOP-007-A
Name vs Quality: Amnesia has the famous name. Bubblegum might be the better smoke. The name sells the first purchase. The experience sells the repeat. We stock what's GOOD, not just what's famous.
Variety Matters: Same strain daily = CB1 receptor downregulation = tolerance. Stock minimum 4 strains, rotate weekly. Educate customers on why variety keeps them feeling good.

SOP-008: Limited Edition Drop Process

Banksy Meets Bauhaus — Edition Management SOP-008-A
  1. Determine edition size (25, 50, or 100 units)
  2. Number each unit: "Edition 23/50"
  3. Create certificate of authenticity (QR code linked)
  4. Document in edition registry (spreadsheet/database)
  5. Photograph each numbered unit before sale
  6. Load into vending machine or bike fleet
  7. Track sales by edition number
  8. Mark sold out when complete
  9. Announce next drop
Scarcity is real: Never reprint a sold-out edition. Copycats can copy product, can't copy edition 23/50 signed by Sylvken.
