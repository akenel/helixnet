# Hotel Pitch Package Template
## What's In The Box

---

## PACKAGE CONTENTS

| # | File | What It Is | For |
|---|------|-----------|-----|
| 1 | `ufa-menu-brochure.pdf` | Menu with pricing | Guest sees options |
| 2 | `ufa-wall-display.pdf` | Wall display + pocket | Hold postcards |
| 3 | `postcard-[hotel]-tent.pdf` | Custom tent card | Stands on desk |
| 4 | `labels-seal-sheet.pdf` | Seal + corner labels | Assembly |
| 5 | `postcard-donny-kenel-STANDARD.pdf` | Sample 6×4" | Show quality |
| 6 | `postcard-donny-kenel-MAX.pdf` | Sample A4 | Show options |

---

## HOW TO USE

1. **Copy this folder** for each new hotel
2. **Customize tent card** with hotel name/branding
3. **Verify all PDFs** open correctly
4. **Send email** using `email-template.md`
5. **Check off** `checklist.md` before sending

---

## FILE TREE

```
[hotel-name]/
├── ufa-menu-brochure.pdf      (standard - don't edit)
├── ufa-wall-display.pdf       (standard - don't edit)
├── postcard-[hotel]-tent.pdf  (CUSTOMIZE THIS)
├── labels-seal-sheet.pdf      (standard - don't edit)
├── postcard-donny-kenel-STANDARD.pdf
└── postcard-donny-kenel-MAX.pdf
```

---

## CUSTOMIZATION

Only the tent card needs customization per hotel:

- Hotel name
- Room number (if applicable)
- Philosophy/tagline
- GPS coordinates
- Brand colors

Everything else is standard UFA branding.
