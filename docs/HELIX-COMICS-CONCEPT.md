## THE BIG IDEA

**What if a 5-year-old could teach a CTO how to fix a toaster?**

Not metaphorically. Literally.

We create comic strips SO SIMPLE that:
1. Kids understand them
2. Kids can DRAW them (LEGO style)
3. Kids explain them to confused adults
4. Adults finally understand HelixNet
5. CTOs stop burning toast

## THE HELIX COLORING BOOK

### Page Ideas:

1. **"Color Leo's Stripes"**
   - Outline of Leo
   - Kids color the stripes
   - Learn: "Leo helps build things"

2. **"Find the Missing Certificate"**
   - Hidden object puzzle
   - Charlie's third eye highlighted
   - Learn: "Always check twice (or three times)"

3. **"Help Pam Scan the Barcodes"**
   - Connect the dots to make barcodes
   - Music notes to color
   - Learn: "Work can be fun with music"

4. **"Fix the CTO's Toaster"**
   - Maze to find the button
   - Toast to color (golden, not burnt!)
   - Learn: "Sometimes the answer is simple"

5. **"Draw Your Own Character"**
   - Blank LEGO person template
   - Kids create their own HelixNet hero
   - Learn: "You can be part of the team"
