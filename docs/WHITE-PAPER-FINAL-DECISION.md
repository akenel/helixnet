# THE FOURTWENTY CSV QUESTION

You're worried about FourTwenty sync. Here's the truth:

**Odoo CAN import CSV files.** Built-in.

```
Odoo → Settings → Import → Upload CSV → Map Columns → Done
```

You can:
1. Use Odoo's native import (free, manual)
2. Schedule automatic imports (needs config, CHF 2-3K)
3. Build Python script like HelixNet has (vibers can do, CHF 5K)

**FourTwenty sync is NOT a reason to keep HelixNet.**
