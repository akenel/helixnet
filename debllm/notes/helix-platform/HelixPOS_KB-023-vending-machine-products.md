## Overview

Artemis Store has a retrofitted vending machine for 24/7 self-service sales. This KB documents product selection, slot allocation, and multi-language requirements.

## Machine Specifications

| Spec | Details |
|------|---------|
| Model | Retrofitted Classic Vending |
| Location | Store entrance |
| Payment | VISA, TWINT (card reader) |
| Age Verification | ID Scanner (18+) |
| Languages | EN, DE, FR, IT (all 4 required) |
| Integration | Standalone - manual entry to Banana |

### Slot Configuration

```
Row A: [A1] [A2] [A3] [A4]     - 4 small slots (cigarette box)
Row B: [  B1  ] [  B2  ]       - 2 large slots (double wide)
Row C: [  C1  ] [  C2  ]       - 2 large slots (double wide)
Row D: [D1] [D2] [D3] [D4]     - 4 small slots (cigarette box)
Row E: [E1] [E2] [E3] [E4]     - 4 small slots (cigarette box)

Total: 12 small + 4 large = 16 product slots
```

|------|------------|------|
