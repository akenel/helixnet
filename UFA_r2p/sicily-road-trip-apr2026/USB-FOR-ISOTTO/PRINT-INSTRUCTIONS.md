## ★ PAPER · CARTA — THIS IS CRITICAL ★

**Use matte cream cardstock, 300gsm.**

> **Italian:** *Carta opaca color crema (o avorio), 300gsm.*
>
> **NOT glossy photo paper.** *NON carta lucida fotografica.*
>
> **NOT thin paper.** *NON carta sottile.*

This deck is meant to feel like an artist's collection, not a glossy holiday photo. **The matte cream finish is the soul of the product.**

If matte cream is not available, fallbacks (in order):
1. Carta opaca avorio 300gsm (matte ivory)
2. Carta opaca naturale 300gsm (matte natural)
3. Carta opaca bianca 300gsm (matte white)

**Avoid at all cost:** carta lucida (glossy), carta fotografica (photo paper), carta < 250gsm (too thin).

## ★ Print Settings · CRITICAL ★

| Setting | Value | Italian |
|---------|-------|---------|
| Paper size | A4 | A4 |
| Orientation | **Landscape** | **Orizzontale** |
| Scale | 100% (no scaling) | 100% (nessun ridimensionamento) |
| Margins | None / 0mm | Nessun margine |
| Color | Full color | Colori |
| Quality | High / Best | Alta qualità |
| **DUPLEX** | **SHORT EDGE FLIP** | **VOLTARE SUL LATO CORTO** |

> ### ⚠ MOST IMPORTANT SETTING ⚠
>
> **DUPLEX: SHORT EDGE FLIP** — *VOLTARE SUL LATO CORTO*
>
> If duplex is set to "long edge", the backs will print upside down behind the fronts. **Always short edge flip for landscape A4 duplex.**

## Cut Dimensions (Cutter Machine — Direct Entry)

**Each sheet uses 2 cuts to make 4 cards.**

| Cut | Direction | Position from edge |
|-----|-----------|-------------------|
| 1 | Horizontal (across the long edge) | **105mm** from top |
| 2 | Vertical (across the short edge) | **148.5mm** from left |

Page size: 297mm × 210mm (A4 landscape).
