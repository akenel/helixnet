## Print Settings

| Setting | Value |
|---------|-------|
| **Paper** | 300gsm matte card stock |
| **Color** | Full color BOTH sides |
| **Duplex** | YES - SHORT EDGE FLIP |
| **Scale** | 100% (do NOT fit to page) |
| **Margins** | None / Borderless |

### CRITICAL: Duplex Setting

```
SHORT EDGE FLIP (Voltare sul lato corto)
```

This ensures the backs align correctly with the fronts when cut.

## Cutting Guide

After printing, each A4 becomes 4 cards with 2 cuts:

### Cut Dimensions for Cutter Machine

| PDF | Page Size | Horizontal Cut | Vertical Cut |
|-----|-----------|----------------|--------------|
| **PORTRAIT** | 210 x 297mm | **148.5mm** | **105mm** |
| **LANDSCAPE** | 297 x 210mm | **105mm** | **148.5mm** |

```
PORTRAIT (210 x 297mm)          LANDSCAPE (297 x 210mm)
┌─────────┬─────────┐           ┌─────────────┬─────────────┐
│    A    │    B    │           │      A      │      B      │
│         │         │           │             │             │
├─────────┼─────────┤ 148.5mm   ├─────────────┼─────────────┤ 105mm
│    C    │    D    │           │      C      │      D      │
│         │         │           │             │             │
└─────────┴─────────┘           └─────────────┴─────────────┘
        105mm                          148.5mm
```

- 3mm white frame is built into each card
- Enter dimensions directly into cutter - no math needed
